# Īss tehniskais apraksts

## Izmantotās tehnoloģijas
- HTML – lapas saturs
- CSS – stils un vizuālais noformējums
- JavaScript – lapas funkcionalitātes papildinājums
- GitHub Pages – publiska lapas izvietošana

## Konfigurācijas faili
- Nav nepieciešami īpaši konfigurācijas faili šim statiskajam projektam.

## Kods
- `index.html` satur struktūru un saturu
- `style.css` satur lapas stilu
- `script.js` satur nelielu funkcionalitāti
